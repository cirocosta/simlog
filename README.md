# Simlog

Simple log messages.

## Install

    npm install simlog --save-dev

## Usage

```javascript
var sl = require('simlog');

sl.sucess('This is a sucess message!');
sl.info('This is an info message!');
sl.warn('This is a warning message!');
sl.error('This is an error message!');
```
